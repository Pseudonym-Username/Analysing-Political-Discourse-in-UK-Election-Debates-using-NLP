This study investigates the extent to which Natural Language Processing (NLP) can be used to analyse political discourse
in UK pre-election televised debates and derive insights relevant to electoral outcomes. To this end, a multi-stage computational
pipeline is developed, incorporating automatic speech recognition, hierarchical topic classification, Large Language Model
(LLM)-assisted annotation, and stance detection framed as a Natural Language Inference (NLI) task. The results show that hierarchical
topic classification models demonstrate improved consistency over baseline approaches, the NLI-based framework provides reliable
stance detection, and when linked to electoral data, derived features exhibit measurable (though not definitive) associations with
working-class vote share. The findings contribute to the growing application of NLP in political science by extending analysis to the
relatively underexplored domain of televised debates and improving understanding of the relationship between communication and
electoral outcomes
